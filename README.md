# neuro-informatics-outcomes
Computational analysis of neurosurgical/neurological outcomes and public health informatics using R and Python. Focused on traumatic brain injury (TBI) and healthcare utilization

This repository serves as a portfolio for clinical informatics projects focused on neurosurgery and public health. My goal is to leverage computational tools (R, Python, Java) to improve patient outcomes and understand healthcare utilization patterns.


### 1. Traumatic Brain Injury (TBI) Trends Analysis
* **Objective:** Analyze longitudinal trends in TBI hospitalizations and discharge dispositions.
* **Tools:** `R`, `Tidyverse`, `ggplot2`.
* **Key Insight:** Identifying demographic "hotspots" for head trauma to inform public health intervention.

### 2. PubMed Research Scraper (Neuro-Informatics)
* **Objective:** Automate literature reviews for specific neurosurgical procedures.
* **Tools:** `Python`, `BeautifulSoup`, `Pandas`.
* **Function:** Extracts metadata from PubMed to identify high-growth areas in surgical research.

## 🛠 Technical Stack
* **Languages:** Python (Pandas, NumPy, Scikit-learn), R (Tidyverse, Shiny, Biostatistics), Java.
* **Methods:** Logistic Regression, Survival Analysis, Machine Learning for Healthcare.
